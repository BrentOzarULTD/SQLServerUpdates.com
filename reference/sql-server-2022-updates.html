Here's the release history for Microsoft SQL Server 2022. Each update is linked to its Microsoft knowledge base article with the download and the list of hotfixes included.

Since SQL Server 2017, we only get Cumulative Updates and GDRs (security fixes), not Service Packs. That makes support end dates simple:

  • Mainstream support ends 2028/01/11
  • Extended support ends 2033/01/11

Note that starting with CU13, there's a patching error for secondary replicas in an AG with databases that have replication, CDC, or SSISDB. For more information about that, read the CU15 release notes. It's left up to you to enable a trace flag or remove databases from an AG before applying CU13 or newer.
